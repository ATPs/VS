The include folder includes most functions.

argProcessor.h, accept and process input commands from the main function.
BFTree.h, construct the binary tree of the cell linages.
HSA.h, most calculation was performed in this file. The HSA class stores TreeS, TreeT, cost score matrix, and calculate the dynamic score matrix.
IOLib.h, seems useless.
NodeClass.h, defines the NodeClass class, which stores the cost score file.
RandomTree.h, generates random tree to calculate pvalue.
SelectCase.h, defines some "macros" of functions, which helps trace the subtrees in the dynamic score matrix.
settings.h, defines some constants in the project.
